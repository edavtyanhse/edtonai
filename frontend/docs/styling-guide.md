# Styling Guide

Этот документ фиксирует ключевые дизайн-токены и градиенты, используемые на лендинге EdTon.ai.

## Цветовые токены

| Токен | Значение | Назначение |
| --- | --- | --- |
| `--gradient-hero` | `linear-gradient(135deg, hsl(217 91% 60%) 0%, hsl(230 91% 70%) 100%)` | Основные CTA и фоновые акценты в хиро-блоке. |
| `--gradient-card` | `linear-gradient(135deg, hsl(0 0% 100%) 0%, hsl(220 14% 98%) 100%)` | Светлый фон для карточек и подложек. |
| `--gradient-aurora` | `linear-gradient(135deg, hsl(217 91% 60% / 0.95) 0%, hsl(261 83% 67% / 0.95) 55%, hsl(191 83% 62% / 0.9) 100%)` | Градиент для блока аналитики в hero и других насыщенных акцентов. |
| `--gradient-insight` | `linear-gradient(145deg, hsl(217 91% 60% / 0.18) 0%, hsl(142 76% 36% / 0.18) 45%, hsl(38 92% 50% / 0.18) 100%)` | Лёгкие фоновые подсветки в карточках с рекомендациями и результатами. |

## Утилитарные классы

| Класс | Описание |
| --- | --- |
| `.gradient-hero` | Применяет `--gradient-hero`. Используется для кнопок CTA и иконок. |
| `.gradient-card` | Применяет `--gradient-card`. Используется для карточек во втором плане. |
| `.gradient-aurora` | Применяет `--gradient-aurora`. Используется в блоках AI-аналитики. |
| `.gradient-insight` | Применяет `--gradient-insight`. Подходит для подложек подсказок и чек-листов. |

> Примечание: все значения указаны в формате HSL и совместимы с темой Tailwind. При добавлении новых токенов сохраняйте форматирование и обновляйте таблицы выше.
